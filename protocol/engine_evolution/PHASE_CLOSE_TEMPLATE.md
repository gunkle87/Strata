# PHASE CLOSE TEMPLATE

Use this template when a phase ends.

## Title
- phase close title

## Phase Purpose
- restate the original phase question

## What Was Proven
- list the concrete findings that were actually established

## What Was Not Proven
- list tempting conclusions that the phase did not justify

## Live Outcome
- what is now live by default
- what was admitted

## Alternate Outcomes
- what remains as alternate, rejected, or deferred

## Baseline Impact
- whether the protected baseline changed
- whether recovery was achieved, lost, or unchanged

## Primary Bottleneck Learned
- the highest-confidence current bottleneck after this phase

## Risks That Remain
- what still needs caution

## Re-entry Criteria
- when this phase or branch should be revisited

## Recommended Next Phase
- one bounded next move, not a broad wishlist
